What is HTML?

HTML ior Hypertext Markup Language as the name suggest is made from two parts.

Hypertext : is the text displayed on web pages and contains links to navigate to other br hypertext documents.A hypertext document may describes objects such as lists,tables,images and forms.

Markup Language:A markup language is a language for annotating documents with
tags that tell a web browser how to display the content on a web page.

Tag name Description
<!DOCTYPE> This tag defines the document type and HTML versions.
<html> This tag encloses the complete HTML document and mainly comprises of document header which is represented by <head>...<head> and document body which is represented by<body>...<body> tags.
<head> This tag represents the document's header which can keep other HTML tags like <title >,<link> etc.
<title> The <title> tag is used inside the <head> tag to mention the document title.
<body> This tag represents the document's body which keeps other HTML tags like <h1> , <div> ,<p> etc.
