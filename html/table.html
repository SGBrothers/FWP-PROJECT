Table

HTML table tag is used to display data in tabular form (row and column). There can be many columns in a row.

We can create a table to display data in tabular form, using <table> element, with the help of <tr> , <td>, and <th> elements.

In Each table, table row is defined by <tr> tag, table header is defined by <th>, and table data is defined by <td> tags.

<caption> Prince </caption>
HTML caption is diplayed above the table. It must be used after table tag only.


Tags

<table>:It defines the table

<tr>:It defines a row in a table

<th>:It defines a header cell in table

<td>:It defines cell in a table

<caption>:It defines table caption


Attributes

Attributes of table tag

Border:You can use border attribute of table tag in HTML to specify border

syntax

<table border =" 4"> Hello <table></table>

colspan:you can use colspan attribute to make a cell span more than one column.

syntax

<th colspan="2">Sourav</th>

rowspan:you can use rowspan attribute to make a cell span more than one row

syntax

<th rowspan="2">Kumar</th>

cellpadding:it defines the whitespace between the cell edge and the content of the cell.

syntax

<table cellpadding="value" >……</table>

cellspacing:it defines the whitespace between the edges of the adjacent cells.

syntax

<table cellspacing="value" >.....</table>
