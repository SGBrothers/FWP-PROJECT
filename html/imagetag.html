Image tag

Image are added to a web page using the <img> element.It is an empty tag which means there is no closing tag for <img> element.

Syntax

<img attribute 1="value" attribute="2" ....>

Attributes of <img> Tag

Attribute Value Description
Src URL Sepcifies the URL of an image
Alt text Sepcifies an alternate text for an image
Align top,bottom,left
middle,left,right 		Sepcifies the alignment of an image according to surrounding elements.
title text Sepcifies title text of image as pop-up
Border pixels Sepcifies the width of the border around an image.
width pixels Sepcifies the width of an image.
height pixels Sepcifies the width of an image.

Example

<src="../images/ocean.png" width="30%" border="4px" title="ocean image" alt="image of ocean">

Image mapping

An image with one or more clickable area is called image map.
Image maps are a valuable way to display complex sets of links.
An image map provides a list of coordinates on a specific image,created in order to hyperlink areas of the image to different destination

For expamle, amn image of solar system,each planet may be hyperlinked to further information about that planet.

HTML elements requried to create image Maps

There are three HTML elements used to create image maps:

img:specifies the location of the image to be included in the map
map: is used to create map of clickable areas.
is used within the map element to define the clickable areas.

Attributes of Area Element

Shape:The shape attribute of area element can have three types of value : rect (rectangle),circle (circle), poly (polygons)

Href:provide the hyperlink target URL for the area

Coords:Specifies the text which will be display on mouse hovering on image.

Alt:provides the alternate text which will be displaced if the image is not visible due top some reason.

Coords:Specifies where to open the target url

example

<map name="ocean" >

<area shape="rect" coords="28,38,251,250" href="https://en.wikipedia.org/wiki/Facebook" title="It will direct you to facebook website" >
<area shape="rect" coords="309,39,550,241" href="https://en.wikipedia.org/wiki/Twitter" title="It will direct you to twitter website" >
<area shape="rect" coords="612,41,831,240" href="https://en.wikipedia.org/wiki/Instagram" title="It will direct you to instagram website" >
