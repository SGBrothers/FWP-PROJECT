Forms

HTML Forms are required, when you want to collect some data from the site visitor.
For example, during user registration you would like to collect information such as name, email address, credit card, etc.

<form action="Script URL" method="GET|POST"> form elements like input, textarea etc. </form>

form attributes

action:Backend script ready to process your passed data.

method:Method to be used to upload data. The most frequently used are GET and POST methods.

Forms control

There are different types of form controls that you can use to collect data using HTML form:

  • text box
  • Radio button
  • Check box
  • Drop down menu
  • text area
  • submit button
  • reset button

input element

The input element is used to insert various controls in the form such as text boxes, password boxes,buttons etc.

Type:It is used to define the type of fields to be created.

The values that this attributes can take are test,password, Radio, Checkbox,etc.

value Description
Text Define a singlew line text field
password Define a password field
checkbox Define a checkbox
radio Define a radio button
submit Define a submit button
reset Define a reset button
Button Define a push button

Name:It is used to specify the name of a field in the form.

value:It is used to specify the default value of a field in the form.

syntax

<input type="text | password | button | radio | checkbox | submit | reset |" name="name of field" value="default value">

TextBox

It is used in situtaions where you eant the visitor to enter only a single line of text

as his name,id,city,etc.

The type attribute of the

<input> element is set to "text" to create a textbox.

syntax

<input type="text" name="text_box">

attribute

size:size Allows to specify the width of the text-input control in terms of characters.

Maxlength:Allows to specify the maximum number of characters a user can enter into the text box.

value:This can be used to provide an initial value inside the control.


Password

When a password is entered, instead of displaying characters,either asterik (*) or dot(.) are shown.

To make textbox that can hide the entered characters,

The type attribute of the <input> element is set to "password"

syntax

<input type="password" name="name of password_box">

Radio button

Radio button is use to dsipaly a small circular button that can be either selected or deslected.
Radio buttons are mutually exclusive i.e. the visitor can choose only one right option from all the choices.

The tye attribute <input> element is set to "radio" to create a radio.

syntax

<input type="radio" name="radio_button_name">

attribute

Name:This attribute is uesd to specify name for the radio button.
The collection of buttons have same name .Thus selecting one will turn the other off
Value:The value that will be used if the radio box is selected.
Checked:Set to checked if you want to select it by default.


CheckBox

The checkBox is used to display small squares that offers the users a number of choices that can toggled "on" and "off".
A tick is displayed if the choice is selected otherwise the square box remains empty
An <INPUT > element is set to "check" to create a cekeckbox.

syntax

<input type="checkbox" name="check_box_name">

attribute

Name:This attribute is uesd to specify name for the radio button.
The collection of buttons have same name .Thus selecting one will turn the other off
Value:The value that will be used if the radio box is selected.
Checked:Set to checked if you want to select it by default.


Submit button

The submit button is used to submit the form data entered by the user to the web server.
An <INPUT > tag with attribute type is set to "submit" to create a submit button

syntax

<input type="submit" value="submit_button_name">

Reset button

The reset button is used to clear all form's data entered by the user and reset it to the initial values.
An <INPUT > tag with attribute type is set to "submit" to create a reset button

syntax

<input type="reset" value="reset_button_name">

Button

The push button is similar in shape to select and reset buttons but like submit and reset buttons, it does not have predefined behaviour.
An <INPUT > tag with attribute type is set to "button" to create a button button

syntax

<input type="button" value="button_name" onclick="function_name()">

Select And Option Tag

It is used to create drop down menus.They allow the user to select a single or multiple choices from a list of choices.
All the choices are not shown on the screen but they are visible when pulls thr drop down list.
That is why they are also known as pull down lists or drop down memus.
The <SELECT > tag is similar in functionality to unordered (<UL>) or ordered lists(<OL>) used in html to define a list and <OPTION> tag is similar to the list item (<LI>) which specifies the list item in a list.

syntax

<select name="select _list_name" multiple> <option value="choice-name1" selected>Text label-1</option> <option value="choice-name2">Text label-2</option> </select>

attribute

Name:Used to give a name to the control which is sent to the server to be recognized and get the value.

SizeThis can be used to present a scrolling list box.

MultipleIf set to "multiple" then allows a user to select multiple items from the menu.


TextArea

The Textarea form control is used if you want the multiple-line information from the visistor.
It enables you to create a field in which th user can enter a large amount of text.
This multiline input area can be implemented using <TEXTAREA>...</TEXTAREA> tags.

syntax

<textarea name="textarea_name" id="" cols="n" rows="m" readonly> Initial content</textarea>

attribute

ROWS:It is a compulsory attribute which is used to set the number of rows of the text that will be visisble without scrolling up or down.

COLS:It is a compulsory attribute which is used to set the number of coloumns of the text that will be visisble without scrolling left or right

Name:It is optional atribute which is used to specifies the name of the textarea field.

ReadonlyIt is optional atribute which when used prevent the ussr from alternating the initial contents.
