Formatting tags


Paragraph tag <p>..</p>

It is paired tag.
It is Block Level Tag.
When a browser displays a paragraph with <p>
element,it usually inserts a new line before the next paragraph and
adds a little bit of extra vertical space.

Syntax

<p>content </p>

Example

<p> this is 1st paragraph </p>

<p> this is 2nd paragraph </p>

Output

this is 1st paragraph

this is 2nd paragraph

Line Break tag <br>

It is unpaired tag.
It is inline Tag.
The <br> element creates Line breaks.When ever you use <br> , anything following it starts on the next line.

Syntax

<br>

Example

<p> this is 1st paragraph <br> this is 2nd paragraph </p>

Output

this is 1st paragraph
this is 2nd paragraph

horizontal tag <hr>

It is paired tag.
It is Block level Tag.
the <hr> element used to draw a horizontal line in your document.
It supports following attributes :


Attributes Description
align specifies the alignment of the horizontal line
size specifies the height of the horizontal line
width specifies the width of the horizontal line
color specifies the color of the horizontal line

Syntax

<hr>

Example

<p> this is 1st paragraph </p>

<hr width="20%" color="green" size="4px>

Output

This is 1st paragraph
