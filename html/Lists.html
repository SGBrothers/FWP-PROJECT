Lists

HTML offers three ways for specifying lists of information


Ordered List <ol>

Ordered list is used to represent your list with some specific squence.
However,you can customize their appearance in browser showing numbers (1,2,3) or alphabets (A,B,C) or roman numerals(i,ii,iii) etc.
Each ordered list is encapsulated in ordered list tags <ol> and each item within the list is nested in the list item tags (<li>)

Syntax

                   <ol>
                        <li> content </li> 
                        <li> content </li> 
                   </ol>

Type and Start are the following attributes of ordered list
Type It is ues to specify the type of numbering you like.
Start:The value of this attribute should be the number from which you want to start the list.

Example

                <ol>
                        <li type="A" start="3"> C language </li> 
                        <li> C++ </li> 
                        <li> python </li>
                </ol>

Output

  1. C Language
  2. C++
  3. Python

UnOrdered List <ul>

UnOrdered list is used when order of the items in the list does not matter.
For example, when ypu want to list flavours of ice cream,you can use Unordered list.
Unordered list is encapsulated in ordered list tags <ul> and each item within the list is nested in the list item tags (<li>)

Syntax

                   <ul>
                        <li> content </li> 
                        <li> content </li> 
                   </ol>

Type and Start are the following attributes of ordered list
Type: It is ues to specify the type of numbering you like.

Example

                <ul type=square>
                        <li> Keyboard </li> 
                        <li> Mouse </li> 
                        <li> printer </li>
                </ul>

Output

  • Keyboard
  • Mouse
  • Printer


Defination List <dl>

The (<dl>) definition list is a special list which provide trems followed by a short text definition or description for them.
The definition list is the ideal way to represent a glossary,list of terms.

Syntax

               <dl>
                    <dt> content </dt> 
                    <dd> content </dd> 
               </dl>

<dl>-Defines the start of the
<dt>-A term
<dd>-Term definition
</dl>-Defines the end of the list

Example

            <dl>
                    <dt> Keyboard </dt> 
                    <dd> It is input device.</dd> 
                    <dt> printer </dt> 
                    <dd> It is output device.</dd>        
             </dl>

Output

keyboard
It is input device
printer
It is output device
